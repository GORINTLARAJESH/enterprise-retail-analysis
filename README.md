# Enterprise Retail Analytics Platform

## Project Objective
Build an enterprise-grade retail analytics solution using Python, PostgreSQL, SQL, and Power BI.

## Tech Stack
- Python
- Pandas
- PostgreSQL
- SQL
- Power BI

## Features
- ETL pipeline
- Data cleaning
- SQL warehouse
- KPI analytics
- Executive dashboard
